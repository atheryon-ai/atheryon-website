# cowork/

Working material for **atheryon-website** — notes, drafts, plans, and docs produced in Cowork
sessions. This is the project's "side thing": committed alongside the code so it syncs
across machines and teammates via GitHub, but kept clearly separate from the code.

## Conventions

- **Text only.** Binaries (PDF, xlsx, images) and anything sensitive do **not** belong
  here — put those in the team drive (Box / OneDrive). Enforced by `.gitignore`.
- **Reference code by repo-relative paths** (e.g. `src/...`, `services/...`), never
  absolute paths like `/Users/you/...`, so links resolve on every device.
- **One subfolder per project/topic**; keep loose files at the root to a minimum.
- It's versioned — commit working docs like any other change. History is a feature.
