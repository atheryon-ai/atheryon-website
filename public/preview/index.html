Seven homepage directions

Visual brainstorm sparked by the vintage CRT/dossier reference. Each variant is a working HTML mockup with real type, real layout, and the locked homepage copy. Click any card to open the full page.

Note — AI-rendered image mockups were blocked on OpenAI org verification, so these are coded HTML wireframes. They're a closer step to production than PNG mockups, but lack photographic texture (CRT scanlines, aged paper) that the reference image used. Pick a direction; the chosen variant becomes the basis for the real site rewrite.
